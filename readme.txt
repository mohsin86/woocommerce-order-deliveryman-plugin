This plugin manages deliveryman for woocommerce Product orders.

== Description ==
This plugin allows to create deliveryman and assign them to orders. 

###Features:
* Manage deliveryman.
* Filter Delivery Man by date and Delivery status.
* Generate report.

== Installation ==
1. Install plugin by searching or uploading its zip.
2. Activate plugin
3. Go to Users->Add New, from WordPress main left menu
4. Create a deliver man (user) and select \'Deliveryman\' from Role drop down
5. Edit any Woo Commerce order and Select Rider from the options in right side and then save order

**Note**: You must have WooCommerce activated before using this plugin.


== Frequently Asked Questions ==
= 1. How do we add order to a delivery man.

Go to user->edit User, assign user role as deliveryman, This user will be shown as delivery man in a order details.
